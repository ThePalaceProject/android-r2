palace-r2-ts
===

This directory contains a Typescript reimplementation of the old Readium JS that runs inside the
web view when displaying an EPUB.

The code is compiled into a minified sr2.js bundle which must then be copied to:

  org.librarysimplified.r2.vanilla/src/main/resources/org/librarysimplified/r2/vanilla/readium/scripts/sr2.js
